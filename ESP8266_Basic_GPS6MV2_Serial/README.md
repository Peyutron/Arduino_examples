## Ejemplo para usar el módulo GPS-6MV2 con un microcontrolador ESP8266.

 # Datos:
* Número de satélites disponibles
* Hdop
* Latitud
* Longitud
* Age
* Fecha
* Hora
* Altitud
* Course
* Velocidad (Km/h
